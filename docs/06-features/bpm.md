# Features: BPM Service (Quy trình)

> Chức năng cần có cho BPM Service
> Cập nhật: 2026-04-24

---

## 📋 Overview

BPM Service quản lý các nghiệp vụ quy trình bao gồm: định nghĩa workflow, instance workflow, user tasks, và cấu hình BPM.

---

## 📊 Danh sách Chức năng

### Định nghĩa Workflow
- Tạo workflow mới
- Sửa workflow
- Xóa workflow
- Import workflow
- Export workflow
- Activate workflow
- Deactivate workflow

### Instance Workflow
- Bắt đầu workflow mới
- Xem instance workflow
- Hủy workflow
- Suspend workflow
- Resume workflow
- Tìm kiếm instance
- Export instance

### User Tasks
- Xem danh sách tasks của user
- Xem chi tiết task
- Complete task
- Assign task
- Delegate task
- Xem lịch sử task
- Tìm kiếm task

### BPM Config
- Cấu hình BPM
- Cấu hình quy trình phê duyệt
- Xem lịch sử workflow
- Export cấu hình BPM
