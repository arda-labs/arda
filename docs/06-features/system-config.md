# Features: System Config Service (Cấu hình Hệ thống)

> Chức năng cần có cho System Config Service
> Cập nhật: 2026-04-24

---

## 📋 Overview

System Config Service quản lý các nghiệp vụ cấu hình hệ thống bao gồm: tham số hệ thống, cấu hình module, lãi suất tham chiếu, ngày làm việc, ngày nghỉ lễ, và đa ngôn ngữ.

---

## 📊 Danh sách Chức năng

### Tham số Hệ thống
- Tạo tham số mới
- Sửa tham số
- Xóa tham số
- Tìm kiếm tham số
- Export tham số

### Cấu hình Module
- Kích hoạt/tắt module
- Cấu hình module
- Xem trạng thái module
- Export cấu hình module

### Cấu hình Chung
- Cấu hình hệ thống chung
- Sửa cấu hình chung
- Export cấu hình chung

### Lãi suất Tham chiếu
- Tạo lãi suất tham chiếu mới
- Sửa lãi suất tham chiếu
- Xem hiệu lực lãi suất
- Xem lịch sử lãi suất tham chiếu

### Ngày làm việc
- Tạo ngày làm việc mới
- Sửa ngày làm việc
- Xóa ngày làm việc
- Import ngày làm việc
- Export lịch làm việc

### Ngày nghỉ lễ
- Tạo ngày nghỉ lễ mới
- Sửa ngày nghỉ lễ
- Xóa ngày nghỉ lễ
- Import ngày nghỉ lễ
- Export lịch nghỉ lễ

### Đa ngôn ngữ
- Thêm bản dịch mới
- Sửa bản dịch
- Xóa bản dịch
- Export bản dịch
- Thiết lập ngôn ngữ mặc định
